## Ordering PCBs
In order to order the most updated PCB from a manufacturer, upload the files under the `CAM` folder to any PCB manufacturing service’s website. Most websites will give various customization options that can typically be left as the default options, though each manufacturer may differ slightly from each other.

## Bill of Materials
The BOM.xlsx includes the description and quantities of the parts used in our design. Some parts that were used were available from the senior design instructors are missing part number information.
